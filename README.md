# Library Management System (Python)
A simple Library Management System built in Python using Object-Oriented Programming (OOP) concepts.
This project allows Admins, Librarians, and Users to manage books, track borrowed books, and store data in files for persistence.

---

Features:-

> Admin

Add new books to the library.
Remove books from the library.
Register new users.

> Librarian

View all registered users.
View list of issued books.
Save all data to files.

> User

Issue a book (if available).
Return a borrowed book.
View all available and issued books.

> Other Features:-

Data persistence using text files (books.txt and users.txt).
Tracks issued books and borrowed books per user.

> Technologies Used:-

Python (Core Language)
OOP Concepts (Classes, Objects, Methods)
File Handling for saving/loading data.
